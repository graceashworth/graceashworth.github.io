Pen&Pal

Interpersonal communication often gets lost in translation when expressed via text message. After moving to New York City to attend Fordham, I began to write my family and friends letters to keep communication open. My love for writing letters, coupled with my obsession for niche stationery, came the creation of Pen&Pen.

Pen&Pen is a modern solution for a not-so-modern act. This IOS social networking application allows users to find pen pals based on similar interests, fostering the possibility of new relationships. I utilized Figma software to design the user interface, following Nielsen's Heuristics of UX-design to create an application that upholds high usability standards for all users. Pen&Pen is developed using Swift Programming Language with a Firebase implemented for secure user authorization and data storage.
